README

Гереатор времени пути в зависимости от координат.
Принимает на вход файл example.txt и формирует два файла
output_data.txt и output_data.csv
в которых содержится информация следующего рода:


[{:type=>0, :coords=>[{:lat=>56.841598, :lon=>35.947228, :time=>1411558991}]}]
(type - тип, coords - координаты и время)
и
imei - идентификатор
time - время прохождения между координатами
lat;lon - координаты
